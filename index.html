Projects

Integra-te

My first project. A very simple website hosted on github pages about the services provided by a friend of mine who is an yoga teacher.
Made only with html and css, not the greatest design but it was a good opportunity to learn to use git and github.

Weather App

No portolio is complete without a weather app. The idea is very simple, just type your location and the app will tell you some data about the weather.
Despite being a simple app, it was a good exercise to learn to work with API's and the setup of a nodejs server. The App makes use of OpenWeatherMap API. which I recommend as a free api to learn and get weather data.

Chat App

An app that serves as a chat room. You can see who's online and send and receibe messages in real time.
This projects leverages the socket.io library that allows to build chat apps in an easy and understable way using sockets. I highly recommend this library to learn more about sockets.

Net Wage / Calculator App

This app is a simple calculator that can also calculate your net wage (for portuguese residents). Currently it doesn't have all the data to compute the wage for all cases but can already be used for tests.
This was a my first attempt at a mobile app using ionic as the tool to convert a web app to a mobile one.

Saving Entities

This Drupal module allows the user to save entities of the type node and media in bulk using a configuration page.
This was my first drupal module and its full compatible with Drupal 9.

Landing Page Scheduler

This module allows the user to set a landing page for a limited duration using a configuration page.
This module is full compatible with Drupal 9.

Bulk Author Update

This module allows the user to update the author of nodes in bulk. The user can configure the "new author" in a configuration page and use the attached view to select which nodes should be update
This module already existed as a Drupal 7 module. I made the port to Drupal 8 trying to keep all the previous features. The module is Drupal 9 ready too.

Select2 Multicheck

This module module provides a new widget for select2 module users. This widget allows multi selection while retaining a dropdown approach.
This module is full compatible with Drupal 9.

Url Restriction By Role

This module provides a simple way for users to configure restrictions in the access of urls using the roles available for the platform.
This module is full compatible with Drupal 9.
